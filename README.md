# Radiansyah Akmal Sofyan — Mahasiswa Teknik Informatika

Portofolio single-page dengan **bento grid**, animasi halus (Framer Motion), dan fokus **case study engineering** (Masalah → Pendekatan → Hasil).

## Beda dari referensi UI/UX

- Tema developer gelap (bukan showcase desain Figma)
- Proyek ditulis seperti mini studi kasus teknis
- Konten mahasiswa TI (kuliah, magang, proyek kampus)

## Mulai

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Kustomisasi

Edit **`src/data/site.ts`** — nama, email, proyek, skill, pendidikan.

## Deploy

Push ke GitHub, lalu deploy gratis di [Vercel](https://vercel.com).
